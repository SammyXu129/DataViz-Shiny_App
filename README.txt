
"Health" fold contain three R files, "global.R" includes data wrangling and EDA, "server.R" contains the instructions that computer needs to build the app, "ui.R" controls the layout and appearance of the app. There are two dataset files and one geological data file in "Health" fold. 

In order to run the Shiny app, before it needs to download library "Shiny", and then go to "Health" file, choose any global.R or ui.R or server.R, click "run app" with green sign to run the app.
